PDF Preview for Odoo

Stop cluttering your Downloads folder! Preview generated PDFs directly in your browser.

❌ The Problem

By default, every time you print a quotation, invoice, or any report in Odoo, it automatically downloads a copy to your local device. This clutters up the Downloads folder with unnecessary duplicate files, causing storage issues and a frustrating user experience.

✅ The Solution

This module intercepts Odoo's native printing behavior. Instead of forcing a download, it instantly opens your PDF report in a new browser tab. You can seamlessly preview the document, and download or print it only if you actually need to using the browser's built-in PDF tools.

Why you need this module

🚀 Instant Preview

No more digging through your file explorer. View reports instantly in a new tab.

💾 Save Storage

Keep your desktop and downloads folder clean from hundreds of temporary invoices and quotes.

⚡ Zero Configuration

Just install the app and it instantly works globally across all Odoo QWeb PDF reports.

Frequently Asked Questions (FAQ)

How do I stop Odoo from downloading PDFs automatically?

By installing the "PDF Preview" module by Odoocrafts, Odoo's default behavior is modified to open PDFs in a new browser tab instead of triggering a forced download.

Does this work for custom Odoo reports?

Yes! Because this module hooks into Odoo's core `ir.actions.report` framework, it automatically applies to all PDF reports, including those generated by third-party or custom modules.

Is it compatible with Odoo 19?

Absolutely. The module is built using the latest Owl JavaScript framework standards ensuring seamless compatibility with Odoo 19.0 Community and Enterprise editions.

Can I still download the PDF if I need to?

Yes. When the PDF opens in a new tab, you can use your browser's native PDF viewer buttons (usually in the top right corner) to save or print the file at any time.
