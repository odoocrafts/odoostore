Frequently Asked Questions (FAQ)

How does this Odoo inventory AI help prevent stockouts?

The AI continuously analyzes your demand trends and supplier lead times. If it detects that projected demand will exceed available inventory, it generates an urgent purchase recommendation before the stockout happens.

Can Inventory Copilot automate Odoo purchasing decisions?

While the MVP provides actionable recommendations that you can review and approve, our goal is to help you automate Odoo purchasing workflows confidently by providing explainable, transparent AI insights.

How do I configure the OpenAI API key?

Go to Inventory > Configuration > Settings. Scroll down to the "Inventory Copilot AI" section and paste your OpenAI API key. The system will use it to power advanced explanations and predictive analytics.

Does it help with dead stock and overstock?

Yes! The dashboard highlights the total value of your dead stock and warns you about products that are overstocked based on recent consumption rates, helping you free up valuable capital.
